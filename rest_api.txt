Application Programming Interface (API)
API adalah sekumpulan fungsi, subroutine, protokol komunikasi, atau kakas / tools untuk membuat perangkat lunak.
interface / antarmuka adalah bagian yang sama antara dua atau lebih komponen yang terpisah pada sebuah sistem kompuer.
definisi API:
- Antarmuka yang berupa kumpulan fungsi yang dapat di-panggil atau dijalankan oleh program lain.
- penerapannya sangat luas
  - Bahasa pemrograman
  - Libary & framework
  - sistem operasi
  - web API / web service

contoh penerapan:
1. Bahasa Pemrograman
______________________________________________________________________________________
// mysql
$mysql = new mysqli("localhost", "root", "root", "mahasiswa");
$result = $mysqli->query("SELECT * FROM mahasiswa");
$row = $result->fetch_assoc();

// PDO
$pdo = new PDO('mysql:host=localhost;dbname=mahasiswa', 'root', 'root');
$statement = $pdo->quary("SELECT * FROM mahasiswa");
$row = $statement->fetch(PDO::FETCH_ASSOC);
_______________________________________________________________________________________
ketika memakai bahasa php ada 2 cara kalau ingin terhubung dengan DBMS mysql, bisa menggunakan sebuah script yang namanya mysqli atau pdo.
API sebuah cara untuk menghubungkan 2 program yang berbeda.
istilahnya adalah koneksi ke database mysql menggunakan 2 API yang berbeda di bahasa pemograman PHP.
